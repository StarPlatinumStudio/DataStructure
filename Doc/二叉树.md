# 二叉树

```Java
class Node{
    private <T> val
       private Node left; //左孩子
       private Node right;//右孩子
    ......
}
```

- 没有根节点的节点叫做子叶节点
- 二叉树具有天然的递归结构
- 左右子树也是二叉树
- 二叉树不一定是‘满的’
- 一个节点甚至NULL也是二叉树

## 二分搜索树

- 是二叉树

- 每个节点的值：

  - 大于其左子树的所有节点值

  - 小于右子树的所有节点值

- 每一颗子树也是二分搜索树
- 存储的元素必须有可比较性
